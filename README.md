# BhromonBilash
“BhromonBilash” is a tour and travel website for making trips across Bangladesh in the palm of your hands.

# to start

```
git clone
```

```Backend
cd backend
npm i
npm start
```

```Frontend
cd frontend
npm i
npm start
```
